【プラグインの説明】

可愛い猫ちゃんがノリノリでリズムを取ります

◆ 使い方 ◆

1. ファイル配置 : ――――――――――――

Windowsを使用している私は
C:\Program Files\Common Files\VST3に配置しました、実際DAWが読み込んでくれるのであれば自分で読み取り先を指定してもらっても構いません。
Macはご自分で調べて配置してください。

2. DAWに読み込ませる : ――――――――

配置が正しければ、DAWでプラグイン管理画面に行き、スキャンなどすれば読み込まれます。

3. 使用方法 : ――――――――――――――

DAWのBPMを自動で読み込んでくれるため、起動したらもう後は見て楽しむのみです。
preリリース版なので上手く猫がリズムを取れないかもです、ご了承ください。
